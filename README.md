# SmartSecure Lockers

A full-stack web application for managing smart locker reservations, payments, and subscriptions.

## Tech Stack

**Frontend:** React, React Router, Axios  
**Backend:** Node.js, Express, MySQL

## Features

- Browse locker locations and available lockers
- Reserve lockers with automatic pricing (hourly rate)
- Payment flow tied to reservations
- Subscription plans management
- Notifications and report generation
- Admin dashboard for managing all resources (users, lockers, reservations, payments, devices, and more)
- Automated reservation expiry job running on server startup
